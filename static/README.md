# Font Awesome に関する注意

この `static` ディレクトリには，Webページのスタイルや動的処理に関連するファイル群が含まれています．

ただし，**Font Awesome のライブラリ一式はデータサイズの都合上，本リポジトリでは追跡対象外（.gitignore）に設定しています**．

そのため，本アプリケーションをクローンした後は，以下の手順で Font Awesome を各自ダウンロードしてください．

---

## 🔧 Font Awesome のセットアップ手順

1. ブラウザで以下の公式ページにアクセスしてください．
   
   [https://fontawesome.com/download](https://fontawesome.com/download)

2. **Version 6** を選択してダウンロードしてください．

3. アカウントをお持ちでない場合は，ダウンロード時に **サインアップ（無料）** が必要になります．

4. zipファイルを解凍して得られたディレクトリの名前を `fontawesome` に修正した後，以下のディレクトリ構成になるように `static` 配下へ配置してください．

```
static/
 └── styles/
      ├── bases/
      ├── pages/
      ├── fontawesome/
      └── ...
```

---

## ⚠ 注意事項

* 本リポジトリには Font Awesome の実体ファイルは含まれていません．
* 正常にアイコンが表示されない場合は，パス設定とファイル配置を確認してください．
* 必ず **Version 6** を使用してください（他バージョンでは互換性の問題が発生する可能性があります）．
